TSKAddShortCutKeyXcodePlugin
============================

ショートカットキーを追加するXcodeのプラグインです。
- Show in Finder -> command + shift + 0
- Push -> command + shift + 7
- History -> command + shift + 6
- Archive -> command + shift + =

# Requirements
- Xcode 5 or later
- ARC

# Usage
1. build (command + B)
2. restart
